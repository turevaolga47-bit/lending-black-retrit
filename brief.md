## 1. ЦЕЛЬ САЙТА

**Первичная цель:** получить заявку (написать в Telegram / заполнить форму).

**Вторичные цели:**
- Сформировать доверие до контакта — через кейсы, процесс, личность
- Объяснить чем я отличаюсь от агентств и других фрилансеров
- Закрыть типичные возражения до созвона

**Что сайт должен делать за 5 секунд:**
1. Ответить: кто это и для кого
2. Показать: это профессионал, а не новичок
3. Дать понятное действие: что делать прямо сейчас

## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

### Основной портрет клиента
**Владелец малого/среднего бизнеса или стартапа**, 28–50 лет.

**Что у него есть:**
- Бизнес, которому нужен сайт / приложение / автоматизация
- Бюджет от 50 000 ₽ на проект
- Негативный опыт с предыдущими разработчиками

**Что болит:**
- Прошлый разработчик пропал после предоплаты
- Сайт есть, но заявок нет — непонятно почему
- Хочу внедрить ИИ, но не знаю с чего начать
- Технари говорят непонятно, менеджеры обещают и не делают

**Как принимает решение:**
- Смотрит кейсы — есть ли похожие задачи
- Читает про процесс — понятно ли, как будет устроена работа
- Ищет живого человека — не агентство, а конкретного исполнителя
- Проверяет социальное доказательство — отзывы, цифры, логотипы

### Вторичная аудитория
**Маркетолог / продакт** в компании, который ищет внешнего подрядчика под конкретную задачу.

## 3. ПОЗИЦИОНИРОВАНИЕ И ОФФЕР

### Главный оффер
> «Делаю цифровые инструменты для бизнеса: сайты, приложения и ИИ-агентов. Фиксированные сроки. Открытые цены. Гарантия правок.»

### Формула оффера
`[Конкретный результат] + [Для кого] + [Дифференциатор] + [Снятие риска]`

**Ключевое отличие от конкурентов:**
- Говорю о результате для бизнеса, а не о технологиях
- Открытые цены — не «напишите для расчёта»
- Три направления в одном: сайты + приложения + ИИ
- Фиксированный процесс с понятными этапами

### 00. НАВИГАЦИЯ

**Расположение:** фиксированная вверху, появляется backdrop-blur при скролле > 60px

**Элементы:**
- Логотип / имя слева: `[Имя] · Цифровой мастер`
- Ссылки: Услуги · Кейсы · Процесс · Обо мне · FAQ
- Кнопка-CTA справа: `Обсудить проект` → ведёт в Telegram

**Поведение:**
- На старте: прозрачная
- При скролле: `background: rgba(10,10,15,.88)` + `backdrop-filter: blur(14px)` + нижняя граница gold
- На мобиле: скрываются ссылки, остаётся лого + кнопка

### 05. ПРОЦЕСС РАБОТЫ

**Цель секции:** снять тревогу «а как это вообще будет?». Прозрачность = доверие.

**Заголовок:**
```
Как устроена работа
```

**6 шагов:**

```
01  Бриф
    Заполняете форму или созваниваемся 30 минут.
    Рассказываете задачу — я задаю уточняющие вопросы.
    [Вы: 30 минут]

02  Оценка
    В течение 24 часов присылаю смету, сроки и план.
    Никаких «подумаем» — конкретные цифры сразу.
    [Я: 24 часа]

03  Договор и старт
    Фиксируем задачи, сроки, стоимость в договоре.
    Предоплата 50% — и приступаю.
    [Вы: предоплата 50%]

04  Разработка
    Работаю итерациями — каждые 3–5 дней показываю прогресс.
    Вы всегда видите, на каком этапе проект.
    [Я: регулярные обновления]

05  Приёмка и правки
    Сдаю результат. Правки до полного принятия — входят в стоимость.
    Никаких «правки за доп. оплату» после сдачи.
    [Вместе: правки включены]

06  Запуск и поддержка
    Запускаем. 30 дней поддержки бесплатно после сдачи.
    Остаток оплаты — только после принятия результата.
    [Вы: оплата 50% + поддержка]
```

### 07. ОТЗЫВЫ

**Цель секции:** убить сомнение «а вдруг он ненадёжный?» голосами реальных клиентов.

**Заголовок:**
```
Что говорят клиенты
```

**Требования к каждому отзыву:**
- Фото клиента (не аватар, реальное)
- Имя + должность + компания
- Конкретный результат, а не «всё хорошо»
- Цифра если есть

**Шаблон отзыва:**
```
«[Конкретный результат]: бот принял 500 заявок за первую неделю,
администратор тратит в 4 раза меньше времени на запись.
Запустились за 10 дней, строго по договору.»

— [Имя Фамилия], [должность], [Компания]
[Фото]
```

**Формат отображения:** карусель или сетка 2×2.

**Если отзывов мало:**
```
- Скриншоты переписки из Telegram/WhatsApp — работают лучше текста
- Видео-отзыв (30 секунд) — +35% доверия
- Логотипы компаний-клиентов без отзывов тоже считаются
```

**Минимум для запуска:** 3 отзыва с фото и именем.

### 09. FAQ

**Цель секции:** закрыть последние сомнения перед тем, как человек уйдёт «подумать».

**Формат:** аккордеон (вопрос → раскрывается ответ по клику).

**10 вопросов:**

```
1. Вы работаете по договору?
   Да, всегда. Фиксируем задачи, сроки, стоимость до старта.
   Предоплата 50%, остаток — после приёмки.

2. Что если мне не понравится результат?
   Правки входят в стоимость и делаются до полного принятия.
   Если расходимся — возвращаю предоплату на условиях договора.

3. Сколько времени займёт мой проект?
   Лендинг — 7–14 дней. Сайт — 21–30 дней. Бот/ИИ — от 7 дней.
   Точные сроки фиксируются в договоре до старта.

4. Вы работаете с маленькими бюджетами?
   Минимальный проект — от [X] ₽. Если бюджет меньше —
   честно скажу и порекомендую другой вариант.

5. Как проходит оплата?
   50% предоплата при подписании договора.
   50% — после сдачи и принятия результата. Безналичный расчёт.

6. Вы делаете всё сами или с командой?
   [Ваш ответ: соло / с проверенными подрядчиками]

7. Что мне нужно подготовить перед стартом?
   Доступы (если есть сайт/соцсети), примеры которые нравятся,
   и понимание задачи. Бриф помогу заполнить сам.

8. Будет ли сайт адаптивным под мобиль?
   Да, всегда. Мобильная версия — обязательная часть любого проекта.

9. Могу ли я потом самостоятельно редактировать сайт?
   Да. При необходимости подключаю CMS и провожу обучение.

10. Почему вы, а не агентство?
    Вы работаете с конкретным человеком, а не с менеджером,
    который передаёт задачу джуну. Прямая связь, личная ответственность.
```

### Шрифты

| Роль | Шрифт | Размер | Начертание |

| Заголовки H1 | Georgia / serif | clamp(2rem, 5.5vw, 3.8rem) | 400 |

```
Темно. Глубоко. Профессионально. Живо.

Не корпоративно — но серьёзно.
Не скучно — но без лишнего шума.
Тёмный фон + золотые акценты = дорого и по делу.
```

### Визуальные принципы

- **Контраст:** текст к фону ≥ 4.5:1 (WCAG AA)
- **Пространство:** большие отступы, воздух между блоками
- **Иерархия:** размер → вес → цвет (не всё сразу)
- **Карточки:** тонкая gold-обводка, rounded corners 16–20px, hover: translateY(-4–6px) + glow

## 7. ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ

### Кнопки

| Тип | Стиль | Текст |
|-----|-------|-------|
| Первичная CTA | `background: linear-gradient(135deg, #c9a84c, #d4829a)` + border-radius 50px + текст dark | «Обсудить проект», «Написать в Telegram» |
| Вторичная | border 1px gold + прозрачный фон | «Смотреть кейс», «Подробнее» |
| Призрачная | только border, hover → фон | «Заполнить форму», «Email» |

**Правило текста кнопки:** глагол + выгода. Не «Отправить» — а «Получить оценку».

**Под главной кнопкой всегда:** снятие риска мелким текстом.

### Формы

**Максимум полей:** 3.
- Имя
- Telegram / телефон
- Кратко о задаче (textarea, необязательно)

**Каждое лишнее поле = -11% конверсии.**

**После отправки:** сообщение «Получил! Напишу в течение часа.»

### Анимации — приоритет реализации

**Срочно (делать сразу):**

| Эффект | Как реализовать |
|--------|----------------|
| Grain overlay | `body::after` + SVG feTurbulence, `opacity: 0.04`, `pointer-events: none` |
| Scroll reveal | IntersectionObserver, `opacity: 0 → 1` + `translateY(28px → 0)`, threshold 0.1 |
| Кастомный курсор | Круг 20px с задержкой следования, увеличивается до 40px на ссылках |
| Typewriter в hero | JS, 4 сменяющихся фразы, скорость 55ms/буква, пауза 2200ms |
| Hover карточек | `transform: translateY(-4px)` + `box-shadow: 0 20px 60px ...` |
| Nav blur при скролле | `backdrop-filter: blur(14px)` при `scrollY > 60` |

**Вторая очередь:**

| Эффект | Как реализовать |
|--------|----------------|
| Kinetic typography | clip-path reveal на H1 при загрузке |
| AI-частицы на фоне | SVG/Canvas, точки + линии, медленное движение |
| Hover-превью проектов | изображение следует за мышью при наведении на список |
| Bento grid для стека | CSS grid с разными размерами плиток |

**Третья очередь (если будет время):**

| Эффект | Как реализовать |
|--------|----------------|
| Horizontal scroll секции кейсов | sticky + translateX |
| Blob morphing в hero | SVG path анимация |
| Variable font animation | font-weight 100→800 при загрузке |

### Иконки

- SVG inline (не шрифты, не PNG)
- Цвет: `stroke: #c9a84c`
- Hover: `filter: drop-shadow(0 0 8px rgba(201,168,76,.9))`
- Размер: 24–32px в карточках, 16px в тексте

### Адаптивность

| Брейкпоинт | Изменения |
|-----------|-----------|
| ≤ 1024px | Карточки 2 колонки вместо 3 |
| ≤ 768px | Все сетки → 1 колонка, nav-ссылки скрыты |
| ≤ 480px | Шрифт body 16px, padding 16px, cursor скрыт |
| ≥ 375px | Минимальный поддерживаемый размер |
